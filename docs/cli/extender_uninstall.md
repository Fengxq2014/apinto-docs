# 卸载拓展插件
### 功能描述
一键化卸载拓展

### 命令
```
./apinto extender uninstall {group}:{project}
```

参数说明：
* group：组织名，如：eolinker.com
* project：项目名称，如：apinto

### 执行示例
```
./apinto extender uninstall eolinker.com:apinto
```
