# 获取拓展可用版本
### 功能描述
根据拓展名获取拓展可用版本

### 命令
```
./apinto extender version {group}:{project}
```

参数说明：
* group：组织名，如：eolinker.com
* project：项目名称，如：apinto

### 执行示例
```
./apinto extender version eolinker.com:apinto
```
