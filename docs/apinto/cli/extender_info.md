# 获取拓展信息
### 功能描述
获取拓展信息，信息包括下载地址和安装命令

### 命令
```
./apinto extender info {group}:{project}:{version}
```

参数说明：
* group：组织名，如：eolinker.com
* project：项目名称，如：apinto
* version：拓展版本号

**注：**当版本号不填时获取latest版本

### 执行示例
```
./apinto extender info eolinker.com:apinto:2.1.0
```
