# 更新拓展插件

### 功能描述
一键化更新拓展，如果本地有该拓展，则直接更新，反之，则从插件市场中下载并更新

### 命令
```
./apinto extender upgrade {group}:{project}:{version}
```

参数说明：
* group：组织名，如：eolinker.com
* project：项目名称，如：apinto
* version：拓展版本号

### 执行示例
```
./apinto extender upgrade eolinker.com:apinto:1.0.0
```
