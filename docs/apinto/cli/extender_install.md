# 安装拓展插件
### 功能描述
一键化安装拓展，如果本地有该拓展，则直接安装，反之，则从插件市场中下载并安装

### 命令
```
./apinto extender install {group}:{project}:{version}
```

参数说明：
* group：组织名，如：eolinker.com
* project：项目名称，如：apinto
* version：拓展版本号

### 执行示例
```
./apinto extender install eolinker.com:apinto:2.1.0
```
