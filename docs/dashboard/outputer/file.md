# 文件输出器

### 功能描述

文件输出器：将请求信息输出到日志文件中，具备以下特性：
* 自定义文件的存放目录及文件名称

* 按照一定周期分割日志文件，避免单个文件过大不好查看的问题

* 定时删除过期文件，降低硬盘空间开销
